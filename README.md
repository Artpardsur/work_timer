## ✨ Возможности

- ⏱️ **Учёт рабочего времени** — запускайте таймер для каждой задачи
- 📊 **Статистика** — анализируйте, сколько времени уходит на разные проекты
- 🖥️ **Два режима работы**:
  - Консольная версия (`main.py`) — для минималистов и автоматизации
  - Графический интерфейс (`gui_app.py`) — для удобного повседневного использования
- 💾 **Сохранение данных** — все записи хранятся локально
- 📅 **История по дням** — просматривайте, чем занимались вчера, на прошлой неделе или в прошлом месяце

## ⚙️ Установка и запуск

### Требования
- Python 3.8 или выше

### Шаги по установке

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Artpardsur/work_timer.git
cd work_timer

# 2. Установите зависимости
pip install -r requirements

# 3. Запустите приложение
# Консольная версия:
python main.py

# Графический интерфейс:
python gui_app.py
```

**Примечание:** Файл зависимостей называется requirements (без расширения .txt). Если вы столкнётесь с ошибкой, переименуйте его в requirements.txt или установите зависимости вручную.

## 📝 Использование

### Консольная версия (main.py)

```bash
# Запуск трекера
python main.py

# Пример команд (если реализованы):
# start "Название задачи" — начать отсчёт
# stop — остановить текущую задачу
# status — показать текущий статус
# report — показать статистику за сегодня
```

### Графический интерфейс (gui_app.py)

```bash
# Запуск трекера
python gui_app.py
```

После запуска откроется окно приложения, где вы сможете:

Ввести название задачи в текстовое поле

Нажать «Старт» — таймер начнёт отсчёт

Нажать «Стоп» — запись будет сохранена

Просмотреть историю — список всех завершённых сессий

Посмотреть статистику — графики и сводки по дням/задачам
